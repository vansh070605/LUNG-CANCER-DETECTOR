About the Project

Lung Cancer Detection is a web application designed to help detect lung cancer based on user input, including symptoms, medical history, and lifestyle factors. The system uses advanced machine learning algorithms to provide an estimated likelihood of lung cancer.

Purpose

The purpose of this project is to create an easy-to-use, accurate, and fast lung cancer detection tool. It aims to raise awareness about lung cancer and help users identify possible symptoms that might require further medical attention.

“Early detection saves lives. Our mission is to provide an easy and effective tool to help users take charge of their health.”

Try the Prediction Tool

How It Works

The application uses data provided by the user, such as age, smoking habits, and specific symptoms to estimate the risk of lung cancer. The backend uses machine learning models trained on large datasets to calculate the probability of lung cancer based on user inputs.

Features

  • Detailed user input form for symptoms, medical history, and demographics.
  • Prediction results with an estimated probability of lung cancer.
  • History tracking of previous predictions.
  • PDF report generation for user reference.
  • Real-time notifications through email and SMS about the results.
